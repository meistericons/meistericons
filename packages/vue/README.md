# @meistericons/vue

## Installtion

### CDN

```bash
<link rel="https://unpkg.com/@meistericons/vue@latest">
```

### npm

```npm
npm install @meistericons/vue --save-dev
```

## Usage

```vue
<script>
import {ArrowDown} from "@meistericons/vue"

export default {
    name:'My Component',
    components: {ArrowDown}
}
</script>

<template>
    <ArrowDown/>
</template>
```
